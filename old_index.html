What is Sipag CMS?

Sipag CMS is single page application to create single page website. Build with Codeigniter and Bootstrap. Simple and rich feature, Sipag allow you to create single page website instantly with standard Bootstrap theme, so you still can easily make your own style.

Features

  • Bootstrap theme
  • Animation support
  • Scrolling navigation
  • Ajax configuration
  • Mailbox

Installation

  • Download and unzip package then extract to your localhost
  • Open you browser, call http://localhost/your-folder-name/install and follow instruction. More information click here
